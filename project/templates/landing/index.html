MAH CET Result 2019

The State Common Entrance Test Cell, Maharashtra will announce the result of MHT CET 2019 on June 3, 2019. MHT CET 2019 result will be declared in online mode which can be accessed by logging in to admission portal using application form, password, and security code.

How to calculate MHT CET 2019 score?

The score of MHT CET 2019 can be calculated by using the below mentioned formula: MHT CET Score = (Total correct answers in Maths x 2) + (Total correct answers in Chemistry & Physics x 1) + 5 marks (allotted by DTE to all candidates)

MHT CET 2019 Facts and Figures

Current year facts and figures of MHT CET 2019 are not available yet. In the previous year, around 2,84,239 candidates have registered for MHT CET 2018 (PCM Group) out of which 2,83,000 had actually appeared in entrance examination for admission to 1,38,741 seats under UG Engineering programmes.

JEE Main related news

JEE Main is applicable for admission to NITs, IIITs and CFTIs participating though Central Seat Allocation Board subject to the condition that the candidate should have secured at least 75% marks in the 12th class examination, or be in the top 20 percentile in the 12th class examination conducted by the respective Boards. For SC/ST candidates the qualifying marks would be 65% in the 12th class examination.
